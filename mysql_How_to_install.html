Οδηγίες Εγκατάστασης ΜySQL Workbench

  1. Κατεβάστε τον installer της αρεσκείας σας (πατήστε εδώ ή για το πλήρες msi σε Windowsεδώ).
  2. Για Windows: Πριν ξεκινήσετε τον installer βεβαιωθείτε ότι έχετε ήδη τα Microsoft .NET Framework 4.5 και Visual C++ Redistributable for Visual Studio 2015(εγκαταστήστε και 32-bit και 64-bit). Για άλλα OS δείτε στη σελίδα τα dependencies.
  3. Για Windows: Ξεκινήσετε τον Installer, πατήστε next σε όλα (μπορείτε να διαλέξετε το developer’s default). Αν κάτι έγινε fail στην εγκατάσταση πιθανότατα δεν έχει εγκαταστήσει αυτά του βήματος 2 (expansions για Excel, Visual Basic, Python κλπ μπορείτε να τα αγνοήσετε αν δεν τα χρησιμοποιείτε).
  4. Για Windows: Στην εγκατάσταση του Server πατήστε σε όλα next. Θα χρειαστεί να ορίσετε ένα κωδικό για το root account (αν θέλετε μπορείτε να διαλέξετε να μην ξεκινά ο Server στο System Startup, αλλά να τον ανοίγετε εσείς όποτε θέλετε με τον MySQL Notifier).
  5. Ανοίξτε το MySQL Workbench και κάντε κλικ στο Local instance MySQL Router (θα ζητηθεί ο κωδικός που ορίσατε στο βήμα 4).
  6. Βάλτε μία μεγάλη τιμή (π.χ. 3000) στο πεδίο που θα βρείτε στο Edit → Preferences → SQL Editor → DBMS connection read time out (in seconds). Στο ίδιο παράθυρο απενεργοποιήστε και το Safe Updates (rejects UPDATEs and DELETEs with no restrictions).
